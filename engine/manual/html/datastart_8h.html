LightDatabase* LightDatabaseCreate ( LightDatabaseDesc &  databaseDesc  ) 

创建一个光线数据库引擎实例

创建一个光线数据库引擎实例，并返回 LightDatabase 指针。

参数:
本函数没有参数。 
评论:
这是在使用数据库引擎前必须要调用的第一个函数，数据库引擎的大部分功能都需要通过调用 LightDatabase 来完成。
返回:
如果函数调用成功，返回一个 LightDatabase 指针，否则返回 NULL 。

释放光线数据库引擎实例

参数:

评论:

返回:
如果函数调用成功，返回一个 LightDatabase 指针，否则返回 NULL 。
