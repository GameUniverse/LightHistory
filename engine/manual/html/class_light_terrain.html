详细描述

地形类

本类负责地形文件的读取，渲染，支持高度图格式的地形。 每个 LightTerrain 实例都必须通过 LightEngine 类的 NewTerrain 方法分配。

void LightTerrain::SetLODFactor ( float  distance,
float  fluctuant 
) [pure virtual]

设置地形显示 LOD 因子

设置地形显示 LOD 因子，此函数用来调节地形显示品质和速度。

参数:
float distance LOD 的距离因子，取值范围为 2.5 ~ 5，此值越大，地形显示品质越高，但显示速度会下降
float fluctuant LOD 的起伏程度因子，取值范围为 25 ~ 50，此值越大，地形显示品质越高，但显示速度会下降
返回:
本函数没有返回值。
