详细描述

角色控制类

本类负责一个角色在物理世界中的正确运算，包括行走、跳跃等绝大部分功能。 每个 LightCharacter 实例都必须通过 LightEngine 类的 createCharacter 函数分配，通过 releaseCharacter 函数释放。

移动角色控制器

移动角色控制器

参数:
const vec3_t vector 是当前帧的位移向量。在一般情况下，当你的角色正在移动时，它是垂直向下的重力和水平移动量（均由此参数指定）的组合。 注意这是一个位移向量（第一次序），它只对当前帧起作用，并且不会累加；它不是一个速度向量（第二次序）或力（第三次序）向量。
float minDist 是提前停止位移递归算法的最小距离值，缺省值是 1。
unsigned long * collisionFlags 是一个位掩码指针，返回在角色移动期间用户要查询的碰撞事件。这是 LightCharacterFlag 标志的组合，它能用来引发各种不同的角色动画。 例如，当此标志为 0 时，就可以播放一个落下的动画，当 LIGHT_CHARACTER_COLLISION_DOWN 标志反回时，就可以播放站立的动画。
float sharpness 此参数保留，供以后使用，必须设置为 1.0f。

返回:
本函数没有返回值。
