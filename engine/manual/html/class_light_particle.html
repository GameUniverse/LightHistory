粒子类

本类负责粒子的创建、管理和显示。每个 LightParticle 实例都必须通过 LightEngine 类的 NewParticle 方法分配。

virtual void LightParticle::follow ( float  factor  )  [pure virtual]

设置粒子追随因子

设置粒子追随因子。通常情况下，粒子离开发射源后，就不再受发射源位置的影响；但总的粒子数是有限的，有时为了更逼真地模拟快速移动的物体， 我们需要让粒子跟着发射源的位置移动，通过设置追随因子，我们就能模拟这种现象。

参数:
float factor 要设置的追随因子，取值范围（0~1），0 表示粒子离开发射源后就不再受发射源位置的影响，1 表示粒子的位置完全由发射源的位置确定。
返回:
本函数无返回值。
