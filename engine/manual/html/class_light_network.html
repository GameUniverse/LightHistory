网络引擎类

本类负责大部分网络系统功能的调用和其它所有网络类的实例分配，是网络引擎最重要的一个类。 此类的实例指针只能通过调用网络引擎创建函数 LightNetworkCreate 得到。举个例子：


LightNetwork * network;
LightNetworkDesc desc;

// 创建光线网络引擎实例
memset( &desc, 0, sizeof(desc) );
desc.hwnd = hwnd;
network = LightNetworkCreate( desc );

// 做一些事情...

// 释放光线网络引擎实例
LightNetworkRelease( network );
