公有属性

mat34_t relPose
 相对于 frameShape（发射器形体）的发射器位置和方向
vec3_t relPosition
 相对于 frameShape（发射器形体）的发射器位置
quat_t relOrientationQuat
 相对于 frameShape（发射器形体）的发射器方向四元数
LightActor * frameShape
 角色指针指向发射器要绑定的角色
int type
 发射器运转模式
int maxParticles
 从这个发射器发射出的最大粒子数
int shape
 发射器的形状可以指定为矩形或椭圆
float dimensionX
 发射器在第一、二轴方向的大小，此轴由它的方向（relPose 或 relOrientationQuat ）决定
float dimensionY
 含义同上
vec3_t randomPos
 发射器方向的每一个轴方向的随机向量值（肯定是一个正值）
float randomAngle
 发射方向的随机角度偏差
float fluidVelocityMagnitude
 发射出去的流体粒子的速度大小
float rate
 这个速度指定每秒钟发射多少个粒子
float particleLifetime
 指定以秒为单位的发射粒子的生命周期
