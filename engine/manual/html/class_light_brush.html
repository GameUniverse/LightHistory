详细描述

画笔类

本类负 2D 和 3D 几何图像的显示，支持 2D 的点、线、矩形、读点， 3D 的点、线、多边形。 每个 LightBrush 实例都必须通过 LightEngine 类的 createBrush 方法分配， releaseBrush 释放。
